# Hosted pages (GitHub Pages)

## Privacy Policy URL (paste in Google Play Console)

https://sajawal07.github.io/Saifi-TV/privacy-policy.html

## Enable Pages (one-time)

1. Open https://github.com/Sajawal07/Saifi-TV/settings/pages  
2. **Source:** Deploy from a branch  
3. **Branch:** `master`  
4. **Folder:** `/docs`  
5. Save — wait 1–2 minutes, then open the URL above.

## Play Console field

**Store listing → Privacy policy** → paste the URL above.
